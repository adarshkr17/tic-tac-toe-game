# Tic Tac Toe Game

A simple and responsive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay
- Winner detection
- Draw detection
- Reset game button
- Clean UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
tic-tac-toe-game/
├── index.html
├── style.css
└── script.js

## How to Run
1. Download or clone the project
2. Open `index.html` in your browser
